> ミニマム作業用画像ビューア

## 特徴

- Slideshow of your favorite images
- Always on Top
- No Title bar
- Easy Moving
- CLI Support

## Download

### OSX

[Download](https://github.com/akameco/yamada/releases)

ダウンロード後、zipを展開し`/Applications`に配置してください。

## CLI

CLIをサポートしています。
ターミナルから現在のディレクトリを簡単に開くことができます。

```
$ yamada .
```

### ヘルプ

```
$ yamada -h
```

## Development

```
$ npm start
```
